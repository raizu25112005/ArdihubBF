# Ardi Hub - Blox Fruits Script

A powerful and feature-rich script for Blox Fruits with anti-ban protection.

## Features

- ⚔️ Auto Farm
  - Level Farming
  - Sea Events
  - Boss Hunting
  - Chest Farming

- 🎯 Auto Quest
  - All Main Quests
  - Submerge Island Quest
  - Event Quests

- 👑 Boss Features
  - Cursed Captain
  - Sea Beast
  - Cake Prince
  - Elite Hunter
  - Sweet Cookie

- 🛡️ Protection
  - Anti-Ban System
  - Anti-Kick
  - Detection Prevention
